# Scoring API

Homework # 2 of the Python Developer Professional Course

## Installation

install redis

```bash
sudo apt install redis-server
sudo systemctl start redis-server
sudo systemctl enable redis-server
```

install module 
```bash
python3 -m venv env
source ./env/bin/activate
pip3 install -e . 
```

## Usage

```bash
scoring_api
scoring_api --help
```
